в config.py токен бота закинуть
запуск с tor браузером
порт может быть 9050 или 9150

гайд по программе:
/start записывает id пользователя в базу данных
по окончанию инициализации базы данных программа оповестит пользователя
/menu выводит 3 инлайн кнопки(создать врага прогресс и настройки)
создать врага:
ожидание ввода имени врага
ожидание фотки
ожидание задачи
прогресс:
вывод уровня
вывод процента выполненых задач, а также процент необходимый для сохранения или перехода на след уровень
вывод задач с возможностью изменить статус выполнения
настройки:
выбор между двумя языками и в будущем возможность изменить задачу
